Создать шаблонную страницу услуги на Next.js App Router - на данном этапе не используем strapi, архитектура fsd

Эта страница будет использоваться как общий шаблон для описания каждой услуги компании. Начнем с financial_diagnostics. Компонент страницы должен принимать данные, и быть готовым отрендерить страницу с содержимым другой услуги

Компоненты секций: 
Header, 
Hero, 
Описание услуги
Как мы работаем
Примеры работ
LeadForm,
FAQ,
Footer

По структуре каждой секции ориентируемся на https://www.tbank.ru/ и https://alfabank.ru/

В дизайне ориентируемся на айдентику в web/DESIGN.md
Для создания новых разделов используй скил web-design-expert

Четко разделяй компоненты

Файлы с данными находятся в web/public/content/ncfg_services.json
Если данные отсутствуют - используем плейсхолдеры
