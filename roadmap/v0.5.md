Создать страницу "Блог" на Next.js App Router - на данном этапе не используем strapi, архитектура fsd

Компоненты секций: Header, Posts, Footer

По структуре каждой секции ориентируемся на https://www.tbank.ru/blog

Поиск и фильтр по тегам реализовывать не нужно

Карточка новости имеет вид 

╭──────────────────────────────────────────────────────────────╮
│  [КАТЕГОРИЯ] • [ДАТА]                                         │
│                                                              │
│  [ЗАГОЛОВОК (крупный)]                                       │
│                    
│                                                              │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│              [Иллюстрация / обложка (16:9)]                   │
│        (картинка на всю ширину, со скруглёнными углами)       │
│                                                              │
╰──────────────────────────────────────────────────────────────╯


В дизайне ориентируемся на айдентику в web/DESIGN.md
Если в DESIGN.md нет информации или есть сомнения в применимости решения - используй скил web-design-expert и дополни web/DESIGN.md

Один файл с данными: web/public/content/blog.json
Сгенерируй несколько постов, структура
{
  title: string
  tags: string[]
  slag: string
  body: html
  anonsImage: string
  createdAt: Data
}

Если данные отсутствуют - используем плейсхолдеры

Четко разделяй компоненты
