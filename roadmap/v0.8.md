--- 

На Post page будет использоватся верстка из админки - обычный html 

Я хочу чтобы ты исопользуя скил web-design-expert и опираясь на текущую дизайн систему DESIGN.md, разработал новый POST_DESIGN.md - соглашение/контракт/дизайн систему для статей в которой прописана стилизация для базовых html элементов <p>, <ul>, <img> и так далее

Затем разработанную систему необходимо применить к странице Post. Моя идея такая - есть базовый контейнер в который будет вставляться верстка из админки, и для всего содержимого контейнера (напрамер .post-content) - применяется разработанный POST_DESIGN для всех элементов
